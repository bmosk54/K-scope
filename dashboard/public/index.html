Zooming out: how the model encodes each tissue type

          
The question: for a whole tissue type, does H-Optimus-0 concentrate the work in a few
             features, or spread it across many? We erase the features most associated with the tissue,
             across 32 tiles of that type, and watch the model’s confidence. Concentrated is the better
             outcome for us: it means a short list of features explains that tissue, and a short list is
             something you can actually look at. Spread out means no such summary exists. It differs by
             tissue type, and that is a fact about the dataset worth knowing.

          Method & scope
          

            
The model’s task. Given one 224×224px tile, pick one of nine tissue types:
            tumour epithelium, lymphocytes, cancer-associated stroma, normal mucosa, mucus, smooth muscle,
            adipose, debris, background. H-Optimus-0 (frozen) encodes the tile; a linear head reads the
            answer out at 99.8% accuracy.

            
What a “feature” is. A sparse autoencoder was trained on 90,000 tile embeddings from
            block 27 — unsupervised, no labels. It learned 6,144 directions and must reconstruct any tile
            using only 40 of them. Those directions are the model’s own visual vocabulary; we did not
            define them.

            
The test. We delete a feature from the live network at block 27 and let the
            remaining 13 blocks run, so the model genuinely recomputes. Every deletion is matched against
            deleting the same number of random features.

            
How this relates to certify. The two answer different questions and are meant to
            be used together. Linear probes are excellent at detecting whether a concept is present in
            the model, which is what certify needs and what probes are built for. The features here are
            what the model computes with, which is what you need if you want to remove something and see
            whether the answer depended on it. Detection and intervention are different jobs. (Measured: a
            99.8%-accurate probe direction, deleted the same way at the same depth, moves the model from 0.999
            to 0.989 — it is a read-out, not a load-bearing part. Deleting the features above moves it to
            0.024.)

            
The two pictures come from two different autoencoders. The feature cards come from the
            whole-tile SAE. The heat map comes from a second SAE fit on 1.6M patch tokens — it shows where
            tissue of the answered type sits, not where card 1’s feature fires. The bases are not
            interchangeable and we do not pretend they are.

            
Limits. These are features the model uses — not validated biomarkers. This
            dataset has no molecular or outcome labels, so nothing here speaks to prognosis. A pathologist
            must confirm the tissue.
